# SalonFlowAI First Message Batch Plan

## Batch Goal

Send the first 10 quality outreach messages with controlled variation.

## Batch Structure

### Message 1-3
Use simple intro style.

### Message 4-6
Use professional business style.

### Message 7-8
Use pain-point style.

### Message 9-10
Use demo invitation style.

## Personalization Rule

Always personalize one line:
- business name
- visible service strength
- local market relevance
- appointment flow observation

## Logging Rule

For every message record:
- date
- channel
- template used
- personalized line
- lead status\n
