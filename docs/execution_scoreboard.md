# SalonFlowAI Execution Scoreboard

## Daily Numbers To Track

- new leads
- first messages
- follow-ups
- warm replies
- demos proposed
- demos booked
- post-demo follow-ups
- offers sent

## Red Zone Warning

If any of these stay at zero for too many days:
- new leads
- outreach
- follow-ups
- demo attempts

then the system is slowing down.

## Green Zone

The pipeline is healthy if:
- leads are added regularly
- outreach happens daily
- follow-ups are consistent
- demos are being proposed
- conversations are moving forward

## Main Principle

Sales momentum comes from volume plus consistency.\n
