# SalonFlowAI First 20 Lead Build Plan

## Goal

Build the first 20 Armenian salon leads in a structured way and move the best ones toward demo.

## Main Objective

By the end of the first lead-building cycle:
- 20 leads identified
- 10 high-priority outreach actions sent
- warm leads identified
- at least 2 demo-worthy businesses found

## Segmentation Mix

- 8 hair salons
- 4 beauty studios
- 3 nail studios
- 3 brow/lash studios
- 2 barbershops

## Main Rule

Do not build the list randomly.
Build it in ranked order with clear scoring and next-step actions.\n
