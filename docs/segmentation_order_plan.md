# Segmentation Order Plan

## First Priority

### Hair Salons
Reason:
- high appointment relevance
- clear service structure
- recurring operational flow

## Second Priority

### Beauty Studios
Reason:
- client records matter
- multiple services
- owner-driven operational pain

## Third Priority

### Nail Studios
Reason:
- repeat appointments
- manageable but useful workflow fit

## Fourth Priority

### Brow / Lash Studios
Reason:
- recurring client cycles
- simpler structure but clear business value

## Fifth Priority

### Barbershops
Reason:
- useful fit, but may vary more by workflow style

## Rule

Attack the highest-likelihood fit first.\n
