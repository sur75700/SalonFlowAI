# Armenia Daily Strike Routine

## Daily Routine

### Step 1 — Find Leads
Add 3 to 5 Armenian salon leads per day.

### Step 2 — Prioritize
Score each lead from 1 to 5.

### Step 3 — Send First Outreach
Send 5 first messages per day.

### Step 4 — Send Follow-Ups
Send 3 to 5 follow-ups per day.

### Step 5 — Move Warm Leads
Push warm replies toward demo.

### Step 6 — Update The Board
Every lead must have:
- status
- next action
- next follow-up date

## End-Of-Day Questions

- How many new leads were added?
- How many first messages were sent?
- How many replies came in?
- How many warm leads moved forward?
- Which channel performed best today?\n
