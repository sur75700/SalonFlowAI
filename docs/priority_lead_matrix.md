# Priority Lead Matrix

## Tier A — Attack First
- score 5 leads
- active business
- visible complexity
- reachable contact
- likely owner pain

## Tier B — Attack Second
- score 4 leads
- good fit
- less obvious urgency
- still strong enough for outreach

## Tier C — Test Pool
- score 3 leads
- possible fit
- useful for message testing

## Tier D — Low Priority
- score 1 to 2 leads
- weak visibility
- unclear fit
- only use if needed

## Main Rule

Protect your time by focusing on Tier A first.\n
