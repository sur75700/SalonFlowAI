# SalonFlowAI First Client Command Path

## Immediate Mission

Turn the first strong lead into the first real paying client.

## Path

1. find strong-fit lead
2. send clear outreach
3. qualify pain
4. push demo
5. show product clearly
6. recommend package
7. ask for next step
8. move toward agreement

## First Client Rule

The first paying client matters more than perfect scale.

## Why

The first client gives:
- proof
- confidence
- testimonial potential
- better closing ability
- better onboarding knowledge\n
