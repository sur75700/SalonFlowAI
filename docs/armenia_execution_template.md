# Armenia Execution Template

## For Each Lead

### Before Outreach
- identify business
- identify strongest channel
- note why they are a fit

### First Message
- choose Armenian script
- personalize opener
- send message
- log date

### If They Reply
- branch using reply scripts
- move to qualification
- propose demo if warm

### If They Do Not Reply
- schedule follow-up
- use second touch
- keep message professional

### After Demo
- log notes
- identify package
- set next close action\n
