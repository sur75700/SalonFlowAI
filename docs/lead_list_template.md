# SalonFlowAI Lead List Template

## Lead Fields

- Business name
- Contact person
- Role
- Instagram
- Facebook page
- WhatsApp / phone
- Email
- City / area
- Business type
- Lead source
- Current booking method
- Pain points
- Interest level
- Last contact date
- Next follow-up date
- Status
- Notes

## Example Status Values

- New
- Contacted
- Replied
- Demo Proposed
- Demo Booked
- Demo Completed
- Follow-Up Needed
- Negotiation
- Closed Won
- Closed Lost

## Qualification Notes

Mark higher priority if the lead:
- actively takes appointments
- has many services
- clearly needs better organization
- mentions admin chaos
- wants stronger reporting\n
