# Armenia WhatsApp Scripts

## Script 1

Բարև ձեզ։ Ես ստեղծել եմ SalonFlowAI հարթակը salon բիզնեսների համար։

Այն օգնում է booking-ները, client-ներին, service-ները, analytics-ը և reporting-ը կառավարել մեկ admin workspace-ում։

Եթե ցանկանաք, կարող եմ ուղարկել կարճ demo։

## Script 2

Բարև ձեզ։ SalonFlowAI-ը salon operation control համակարգ է, որը օգնում է salon owner-ներին booking-ները, հաճախորդների բազան, service control-ը և reporting-ը ավելի կազմակերպված վարել։

Եթե հետաքրքիր լինի, կարող եմ կարճ ներկայացնել։
