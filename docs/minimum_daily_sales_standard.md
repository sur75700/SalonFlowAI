# SalonFlowAI Minimum Daily Sales Standard

## Minimum Standard

Every day must include at least:
- 3 new leads
- 3 first outreach messages
- 2 follow-ups
- board update
- next actions assigned

## Strong Day Standard

A strong day includes:
- 5+ new leads
- 5+ first outreach messages
- 3+ follow-ups
- 1 warm conversation
- 1 demo push

## Main Rule

Consistency beats random intensity.\n
