# Armenia Facebook Outreach Scripts

## Script 1

Բարև ձեզ։ Ես ստեղծել եմ SalonFlowAI հարթակը salon բիզնեսների համար։

Այն օգնում է booking-ները, client-ներին, service catalog-ը, analytics-ը և reporting-ը մեկ admin համակարգի մեջ կառավարել։

Եթե հետաքրքիր լինի, կարող եմ ուղարկել կարճ demo preview։

## Script 2

Բարև ձեզ։ Ես աշխատում եմ salon operation control համակարգի վրա, որը կարող է օգնել salon-ներին ավելի մաքուր կազմակերպել իրենց booking-ները, client records-ը, service control-ը և daily reporting-ը։

Եթե ցանկանաք, կարող եմ կարճ ներկայացնել՝ ինչպես է աշխատում SalonFlowAI-ը։

## Script 3

Բարև ձեզ։ Կապ եմ հաստատում, որովհետև ստեղծել եմ SalonFlowAI անունով համակարգը salon բիզնեսների համար։

Այն նպատակ ունի scattered admin աշխատանքը բերել մեկ հարթակ՝ bookings, clients, services, insights և reports։

Եթե հետաքրքիր լինի, կարող եմ ուղարկել demo։\n
