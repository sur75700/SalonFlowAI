# SalonFlowAI Daily Execution Routine

## Morning
- check replies
- update statuses
- identify warm leads
- schedule follow-ups

## Midday
- send new outreach
- personalize messages
- log all actions

## Afternoon
- handle replies
- move conversations toward demo
- send summaries if needed

## End Of Day
- update tracker
- count outreach sent
- count follow-ups sent
- count warm replies
- count demos proposed
- set tomorrow priorities

## Daily Non-Negotiables

- add leads
- send outreach
- send follow-ups
- update statuses
- push at least one lead forward\n
