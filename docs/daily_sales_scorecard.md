# SalonFlowAI Daily Sales Scorecard

## Daily Metrics

- New leads found
- First outreach messages sent
- Follow-ups sent
- Replies received
- Qualified leads
- Demo invitations sent
- Demos booked
- Demos completed
- Offers sent
- Wins
- Losses

## Weekly Questions

- Which outreach channel performed best?
- Which message got the best replies?
- Which objections appeared most often?
- Which package was easiest to position?
- Where did leads get stuck most often?

## Core Goal

Turn outreach into a repeatable system instead of random effort.\n
