# SalonFlowAI Demo Talking Points

## Opening
SalonFlowAI is a modern salon admin control center built to unify bookings, clients, services, insights, and reporting in one place.

## Demo Flow
1. Show Dashboard
   - Explain executive overview
   - Show key metrics: clients, services, bookings, completed, scheduled, today

2. Show Bookings Control
   - Explain appointment creation
   - Explain status flow and daily booking management

3. Show Client Registry
   - Explain client creation and search workflow
   - Explain why organized customer data matters

4. Show Service Catalog
   - Explain service pricing, duration, and activation state
   - Explain operational consistency

5. Show Insights
   - Explain completed revenue
   - Explain scheduled pipeline
   - Explain cancelled value
   - Show executive snapshot

6. Show PDF Reports
   - Explain export readiness
   - Explain daily summary reporting

7. Show Workspace
   - Explain operator shortcuts
   - Explain backend access and admin utility role

## Closing
SalonFlowAI is not just a UI. It is an operating layer for salon businesses.
