# Armenia Channel Execution Board

## Instagram Board

Track:
- leads found
- messages sent
- replies
- warm replies
- demos proposed

## Facebook Board

Track:
- pages contacted
- messages sent
- replies
- warm replies
- demos proposed

## WhatsApp Board

Track:
- direct outreach count
- follow-up count
- demo conversations
- strong prospects

## Email Board

Track:
- emails sent
- opens if known
- replies
- demo requests

## Main Rule

Do not mix all channels without tracking.
Know which channel is bringing:
- replies
- warm leads
- demo movement\n
