# SalonFlowAI Sales KPI Board

## Core KPIs

- Leads Added
- First Messages Sent
- Follow-Ups Sent
- Reply Rate
- Warm Lead Rate
- Demo Proposal Count
- Demo Booking Count
- Demo Completion Count
- Offer Count
- Close Count

## Health Indicators

### Healthy Pipeline
- leads are added regularly
- outreach happens daily
- follow-ups are consistent
- demos are being proposed
- warm replies exist
- offers move forward

### Weak Pipeline
- no new leads
- inconsistent outreach
- weak follow-up
- no demos
- no next actions
- no close movement

## KPI Goal

Use KPIs to control action, not just to observe numbers.\n
