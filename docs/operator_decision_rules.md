# SalonFlowAI Operator Decision Rules

## Rule 1
If the lead score is high, outreach faster.

## Rule 2
If the lead replies warmly, move to demo quickly.

## Rule 3
If the lead is unclear, ask qualification questions before overselling.

## Rule 4
If the lead is cold, do not waste too much time too early.

## Rule 5
If the demo goes well, never disappear after it.

## Rule 6
If there is a strong fit, recommend the package directly.

## Rule 7
If the pipeline slows down, increase lead intake before anything else.

## Rule 8
If responses are weak, improve openers and channel targeting.

## Rule 9
If demos happen but closes do not, improve package fit and close language.

## Rule 10
If one channel wins, double down on it.\n
