# SalonFlowAI First 3 Days Action Plan

## Day 1 — Lead Foundation

### Main Goal
Build the first strong lead base.

### Tasks
- find 10 Armenian salon leads
- score each lead from 1 to 5
- record business name, channel, and notes
- identify top 5 priority leads
- prepare first message batch

### End Of Day Output
- 10 leads added
- 5 top-priority leads selected
- first outreach plan ready

## Day 2 — First Outreach Push

### Main Goal
Start outbound movement.

### Tasks
- send first outreach to 5 leads
- personalize each opener
- log date and message version
- prepare follow-up dates
- record first reactions

### End Of Day Output
- 5 first outreach messages sent
- statuses updated
- next follow-up dates assigned

## Day 3 — Expansion And Follow-Up

### Main Goal
Build momentum.

### Tasks
- add 5 more leads
- send 5 more first outreach messages
- follow up with day 2 leads if appropriate
- identify warm leads
- push at least 1 lead toward demo

### End Of Day Output
- 15 total leads
- 10 first outreach messages sent
- warm leads identified
- first demo candidate selected\n
