# SalonFlowAI Daily Command Center

## Daily Command Targets

- new leads added
- first outreach sent
- follow-ups sent
- warm replies handled
- demos proposed
- demos booked
- offers advanced

## Daily Operating Blocks

### Block A — Lead Intake
Add and rank new leads.

### Block B — First Outreach
Send first messages with correct scripts.

### Block C — Follow-Up
Send all required follow-ups for existing leads.

### Block D — Warm Lead Handling
Reply fast and move toward demo.

### Block E — End-of-Day Control
Update board, next actions, and priorities.

## Non-Negotiables

Before ending the day:
- every touched lead has a next action
- every warm lead has fast follow-up
- pipeline status is updated\n
