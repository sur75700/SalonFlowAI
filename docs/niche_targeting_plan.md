# SalonFlowAI Niche Targeting Plan

## Primary Niche

Salon and beauty service businesses.

## Sub-Niches

- hair salons
- beauty studios
- nail studios
- barbershops
- brow / lash studios
- mixed beauty centers

## Best First Buyers

The best first buyers are businesses with:
- recurring appointments
- multiple services
- owner-led operations
- client record needs
- visible admin complexity

## Positioning By Niche

### Hair Salons
Use language around bookings, service timing, pricing control, and daily flow.

### Nail / Beauty Studios
Use language around appointment control, client records, and cleaner admin workflow.

### Barbershops
Use language around booking flow, service clarity, and operational speed.

### Multi-Service Beauty Centers
Use language around service catalog visibility, reporting, and centralized control.

## Main Targeting Rule

Sell operational clarity, not only software.\n
