# Founder Daily Money Routine

## Morning
- review pipeline
- review warm leads
- review follow-ups due
- choose top money priority for the day

## Midday
- send outreach
- answer replies
- move leads toward demo
- update statuses

## Afternoon
- run demos or push demos
- make package recommendations
- follow up on open opportunities

## End Of Day
- update money dashboard
- count progress
- identify bottleneck
- set tomorrow's strongest money task

## Rule

Every day should create visible movement toward cash.\n
