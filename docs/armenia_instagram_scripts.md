# Armenia Instagram Outreach Scripts

## Armenian Script 1 — Simple Intro

Բարև ձեզ։ Ես ստեղծել եմ SalonFlowAI համակարգը salon բիզնեսների համար։

Այն օգնում է մեկ admin workspace-ի մեջ կառավարել booking-ները, client-ներին, service-ները, analytics-ը և reporting-ը։

Եթե հետաքրքիր լինի, կարող եմ ուղարկել կարճ demo preview։

## Armenian Script 2 — More Professional

Բարև ձեզ։ Ես աշխատում եմ salon բիզնեսների համար admin control համակարգի վրա, որը կոչվում է SalonFlowAI։

Այն նախատեսված է booking-ները, client records-ը, service control-ը, insights-ը և PDF reporting-ը մեկ հարթակի մեջ միավորելու համար։

Եթե ցանկանաք, կարող եմ ուղարկել կարճ demo և ներկայացնել՝ ինչպես դա կարող է օգտակար լինել ձեր salon-ի workflow-ի համար։

## Armenian Script 3 — Pain Point Focused

Բարև ձեզ։ Ստեղծել եմ SalonFlowAI հարթակը այն salon-ների համար, որոնք ուզում են booking-ները, client-ները, service-ները և reporting-ը ավելի կազմակերպված կառավարել։
