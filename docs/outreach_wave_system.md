# Outreach Wave System

## Wave 1 — Top Priority Leads
Send first outreach to the strongest 5 leads.

## Wave 2 — Strong Supporting Leads
Send outreach to the next 5 leads after reviewing early responses.

## Wave 3 — Broader Test Leads
Use the remaining 10 leads to test message quality, fit, and response patterns.

## Why This Matters

This avoids burning all leads with weak messaging at once.
It lets you improve your scripts before hitting the full list.

## Wave Logic

- Wave 1 = best-fit businesses
- Wave 2 = strong backups
- Wave 3 = broader test pool\n
